# 311 Knowledge Base Ingestion Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Pull ~1000 Philly 311 knowledge base articles from a REST API, convert their Salesforce Knowledge HTML to clean markdown, and index them into a new pgsearch `knowledge-311` index in dev so we can evaluate search quality via the existing manual search page.

**Architecture:** One file — `apps/api/scripts/ingest-311-kb.ts` — composed of pure exported functions (`fetchArticleList`, `fetchArticle`, `transform`, `ensureIndex`, `pushDocument`, `refreshIndex`) plus a bottom-of-file `main()` CLI driver. Functions wrap `@phila/pgsearch-client` for pgsearch calls and the `@phila/search-parse` pipeline for HTML→markdown. Each function is independently importable so a future scheduled runner can reuse them without edits. Sequential detail fetches; no retries; one unit test file covering the HTML transform against fixture articles.

**Tech Stack:** TypeScript, `tsx`, vitest, `@phila/search-parse` (HTML→markdown), `@phila/pgsearch-client` (pgsearch HTTP client).

**Spec:** `docs/superpowers/specs/2026-04-13-311-kb-index-design.md`

**Spec deviations (all documented in-plan):**
- Test file lives at `apps/api/test/ingest-311-kb.test.ts` (not co-located with the script) to match the vitest pattern `test/**/*.test.ts` defined in `apps/api/vitest.config.ts`.
- Fixtures live at `apps/api/test/fixtures/311-kb/` (no pre-existing `__fixtures__` convention in the repo).
- `PGSEARCH_ADMIN_KEY` is **optional**. In local dev admin auth is unenforced (`apps/api/scripts/bootstrap-dev-index.ts` makes admin calls with no auth header). If the dev API turns out to enforce a key, setting the env var lights it up with no code change.
- `transform()` returns `null` (never throws) for malformed or empty articles. Spec left the choice open; picking `null` unifies "skipped" accounting.

```typescript
export type RawArticleListItem = {
  id: string
  title: string
  lastPublishedAt: string
  url: string
}

export type RawArticle = RawArticleListItem & {
  content: string
}

export type ArticleListPage = {
  articles: RawArticleListItem[]
  nextLink: string | null
}

const LINK_NEXT_RE = /<([^>]+)>;\s*rel="next"/

export async function fetchArticleList(
  base: string,
  apiKey: string,
  offset: number,
  limit: number,
): Promise<ArticleListPage> {
  const url = `${base}/private/key/knowledge-articles?offset=${offset}&limit=${limit}`
  const res = await fetch(url, { headers: { 'x-api-key': apiKey } })
  if (!res.ok) {
    throw new Error(`list fetch failed: ${res.status} ${await res.text()}`)
  }
  const body = (await res.json()) as { articles: RawArticleListItem[] }
  const linkHeader = res.headers.get('link') ?? ''
  const match = LINK_NEXT_RE.exec(linkHeader)
  return { articles: body.articles, nextLink: match ? match[1] : null }
}

export async function fetchArticle(
  base: string,
  apiKey: string,
  id: string,
): Promise<RawArticle> {
  const res = await fetch(`${base}/private/key/knowledge-articles/${id}`, {
    headers: { 'x-api-key': apiKey },
  })
  if (!res.ok) {
    throw new Error(`detail fetch failed for ${id}: ${res.status} ${await res.text()}`)
  }
  return (await res.json()) as RawArticle
}

export async function* iterateArticleIds(
  base: string,
  apiKey: string,
): AsyncGenerator<RawArticleListItem> {
  let offset = 0
  const limit = 50
  while (true) {
    const page = await fetchArticleList(base, apiKey, offset, limit)
    for (const a of page.articles) yield a
    if (!page.nextLink) return
    // Parse offset from the next link to avoid assuming fixed-stride pagination.
    const nextOffset = Number(new URL(page.nextLink).searchParams.get('offset'))
    if (!Number.isFinite(nextOffset) || nextOffset <= offset) return
    offset = nextOffset
  }
}
```

```typescript
import { pipeline, cleanWhitespace, toMarkdown } from '@phila/search-parse'

const parseKbHtml = pipeline(cleanWhitespace(), toMarkdown())

export async function transformAsync(raw: RawArticle): Promise<IngestDocument | null> {
  let parsed
  try {
    parsed = await parseKbHtml(raw.content ?? '')
  } catch (err) {
    console.warn(`[ingest-311-kb] parse failed for ${raw.id}:`, err instanceof Error ? err.message : err)
    return null
  }
  const body = parsed.body.trim()
  if (body.length === 0) return null
  return {
    external_id: raw.id,
    title: raw.title,
    body,
    metadata: {
      source: 'phila-311-kb',
      source_slug: raw.url,
      source_url: `${ARTICLE_URL_BASE}${raw.url}`,
      last_published_at: raw.lastPublishedAt,
    },
  }
}
```

```typescript
import { PgsearchClient, type IndexInfo, type IngestResponse } from '@phila/pgsearch-client'

export type EnsureIndexResult =
  | { created: true; index_key: string; search_key: string; info: IndexInfo }
  | { created: false; info: IndexInfo }

export async function ensureIndex(
  client: PgsearchClient,
  name: string,
): Promise<EnsureIndexResult> {
  try {
    const info = await client.getIndex(name)
    return { created: false, info }
  } catch (err) {
    // Expected on first run — index does not exist yet.
  }
  const created = await client.createIndex({
    name,
    description: 'Philly 311 knowledge base articles (Salesforce Knowledge export)',
  })
  return {
    created: true,
    index_key: created.index_key,
    search_key: created.search_key,
    info: created,
  }
}

export async function pushDocument(
  client: PgsearchClient,
  indexName: string,
  indexKey: string,
  doc: IngestDocument,
): Promise<IngestResponse> {
  return client.ingest(indexName, doc, indexKey)
}

export async function refreshIndex(
  client: PgsearchClient,
  indexName: string,
): Promise<void> {
  await client.refreshIndex(indexName)
}
```

**Note on `ensureIndex` error handling:** the `getIndex` 404 path is "expected." If the client throws on 404, catch and fall through to create. If it returns `null` or an `{ok: false}` discriminator, adjust the control flow to match. **Do not wrap real errors (500s, network failures) in the "must not exist" branch** — check the error shape and rethrow anything that isn't a clean 404.         `[ingest-311-kb] index '${INDEX_NAME}' already exists but KNOWLEDGE_311_INDEX_KEY is not set.`,
```typescript
      )
      console.error(`[ingest-311-kb] keys are bcrypt-hashed and unretrievable; use the value from the first-run banner.`)
      process.exit(1)

  console.log(`[ingest-311-kb] catalog size: ${total}`)

  let indexed = 0
  let skipped = 0
  let failed = 0
  const startedAt = Date.now()

  for (let i = 0; i < total; i++) {
    const item = ids[i]
    try {
      const raw = await fetchArticle(env.kbApiBase, env.kbApiKey, item.id)
      const doc = await transformAsync(raw)
      if (doc === null) {
        skipped++
        console.warn(`  skip ${item.id}: empty body`)

        await pushDocument(client, INDEX_NAME, indexKey, doc)
        indexed++
      }
    } catch (err) {
      failed++
      console.warn(`  skip ${item.id}: ${err instanceof Error ? err.message : err}`)
    }
    if ((i + 1) % 50 === 0) {
      console.log(`[${i + 1}/${total}] indexed=${indexed} skipped=${skipped} failed=${failed}`)
    }
  }

  console.log(`[ingest-311-kb] refreshing index`)
  try {
    await refreshIndex(client, INDEX_NAME)
  } catch (err) {
    console.warn(`[ingest-311-kb] refresh failed (documents still indexed):`, err instanceof Error ? err.message : err)
  }

  const elapsedSec = ((Date.now() - startedAt) / 1000).toFixed(1)
  console.log(``)
  console.log(`[ingest-311-kb] done in ${elapsedSec}s`)
  console.log(`  total:   ${total}`)
  console.log(`  indexed: ${indexed}`)
  console.log(`  skipped: ${skipped}`)
  console.log(`  failed:  ${failed}`)
}
```
